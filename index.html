Getting Started

1

Subscribe to Event Sentinel

Choose a subscription plan that fits your needs. All plans include fully managed hosting - we handle all the technical setup, maintenance, and updates for you. View pricing plans.

2

Create Discord Bot & Provide Token

Go to the Discord Developer Portal and create a new application. Add a bot to your application and enable these Privileged Gateway Intents: Presence Intent, Server Members Intent, and Message Content Intent. Copy your bot token and provide it to us through your subscriber portal. We'll use this token to set up your custom bot instance.

3

Bot Setup & Activation

Once we receive your bot token, we'll set up your custom Event Sentinel bot instance on our servers. This typically takes 24-48 hours. You'll receive a notification when your bot is ready, along with an invite link to add it to your Discord server.

4

Invite Bot to Your Server

Click the invite link we provide and authorize the bot with required permissions: View Channels, Send Messages, Embed Links, Attach Files, Read Message History, and View Audit Log. The bot will automatically join your server and start logging events.

5

Configure Log Channel

Use the /setlogchannel command in your Discord server (admin only) to select where events will be logged. The bot is already online and ready to use - just configure your preferences!

Usage Guide

Commands

/setlogchannel <channel>

Set the log channel for event logging (Admin only). Select the channel where you want events to be logged for this specific server.

Using the Bot

The bot is already online and running on our servers 24/7. Once you've invited it to your server and configured the log channel, it will automatically start logging events. No need to run any commands or manage the bot - we handle all that for you!

Configuration

Configure your log channel using /setlogchannel in your Discord server. The bot supports per-server configuration, so each server can have its own log channel. All settings are managed through Discord commands - no file editing required!

Getting Help

For issues, check: bot permissions, required intents are enabled, log channel is configured, and bot can send messages in the log channel. Visit the GitHub repository for detailed documentation and support.
